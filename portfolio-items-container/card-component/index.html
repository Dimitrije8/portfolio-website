Live Site : Product preview card component

Overview:

The goal of this project was to design a website to improve user experience, increase conversion rates and modernize the visual design.

Process:

1. Research & Discovery:

  • Analyzed websites to identify best practices and areas for improvement.
  • Created user personas and user journey maps to guide the design process.

2. Wireframing & Prototyping:

  • Developed low-fidelity wireframes to outline the website's structure and layout.
  • Iterated on wireframes based on feedback from stakeholders and usability testing.
  • Created interactive prototypes to simulate user interactions and test navigation flows.

3. Visual Design:

  • Defined a modern and visually appealing design language based on brand guidelines.
  • Designed high-fidelity mockups incorporating typography, color scheme, and imagery.
  • Ensured consistency across all screens and elements to maintain a cohesive user experience.

Built with:

  • Semantic HTML5 markup
  • CSS custom properties
  • CSS Variables
  • CSS Flexbox/Grid
  • JavaScript
